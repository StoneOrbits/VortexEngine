24 //
25 // This file is the interface for the Vortex Engine.
26 //
27 // The things in Vortex that you can hook into include:
28 //
29 // - The serial input/output callbacks (editor connection)
30 // - The Wireless IR/VL input/output callbacks (mode sharing)
31 // - The digital pin read callbacks (buttons press)
32 // - The leds init, brightness, and display (render the lights)
33 //
34 // Then on top of being able to hook into the various actions of the engine
35 // the Vortex class provides all kinds of APIs for controlling and manipulating
36 // the engine while it is running.
37 //
