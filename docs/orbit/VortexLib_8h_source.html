20 //
21 // This file is the interface for the Vortex Engine.
22 //
23 // The things in Vortex that you can hook into include:
24 //
25 // - The serial input/output callbacks (editor connection)
26 // - The Wireless IR/VL input/output callbacks (mode sharing)
27 // - The digital pin read callbacks (buttons press)
28 // - The leds init, brightness, and display (render the lights)
29 //
30 // Then on top of being able to hook into the various actions of the engine
31 // the Vortex class provides all kinds of APIs for controlling and manipulating
32 // the engine while it is running.
33 //
